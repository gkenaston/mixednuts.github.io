Match Scheduling

How matches are created, edited, and canceled.

Recurring Schedules

When you create a group, you set a recurrence pattern -- for example, every Tuesday at 7:00 PM, or every other Saturday at 10:00 AM. Mixed Nuts uses this pattern to automatically generate matches for upcoming dates within a rolling window.

Supported recurrence types include:

  • Weekly (e.g., every Monday)
  • Biweekly (e.g., every other Friday)
  • Custom patterns via recurrence rules (e.g., first and third Wednesday of each month)

One-Off Matches

Group owners and admins can create additional matches outside the recurring schedule. This is useful for makeup games, tournaments, or special events. One-off matches appear alongside recurring ones on the group screen.

Editing a Match

Owners and admins can edit match details such as time, location, and max players. When editing a recurring match, you can choose to:

  • Edit this match only -- changes apply to just this occurrence
  • Edit all future matches -- changes apply to this and all subsequent occurrences

Canceling a Match

Owners and admins can cancel a match. When a match is canceled, all opted-in players receive a push notification and email alert informing them of the cancellation.

Match Details

Each match displays:

  • Scheduled time -- in the group's timezone
  • Location -- where the match will be played
  • Max players -- the cap on confirmed roster spots
  • Current roster -- confirmed players and reserves
