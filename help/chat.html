Help / Group Chat

Group Chat

Stay connected with your group between matches.

Sending Messages

Each group has a built-in chat. Open the group and navigate to the Chat tab to view and send messages. Type your message and tap Send. Messages are delivered in real time to all group members.

Who Can See Messages

All members of the group can see all messages in the group chat. There are no private channels or direct messages within Mixed Nuts -- the chat is a shared group space.

Deleting Messages

You can delete your own messages by long-pressing (or swiping) on the message and selecting Delete. Group owners and admins can delete any message in the group chat to moderate content.

Muting Chat Notifications

If a group chat is too active, you can mute notifications for that group. Muting stops push notifications for new chat messages but does not affect other notifications (roster changes, match updates, etc.). You can unmute at any time from the group settings.

Unread Count Badge

The group list shows an unread count badge next to groups with new chat messages you have not yet seen. Opening the chat clears the badge. This helps you quickly see which groups have new activity.
